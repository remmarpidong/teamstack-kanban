# TeamStack Kanban

AI-Powered Kanban Board for development teams.

## Getting Started

```bash
npm install
npm run dev
```

## Features

- Drag and drop tasks between columns
- AI agent integration (via OpenClaw)
- Real-time updates
- Dark mode

## Tech Stack

- Next.js 14
- React
- Tailwind CSS
- @hello-pangea/dnd (drag and drop)
- OpenClaw API
